Linear Search:

    Algorithm:

        1)  Start
        2)  Set Values
                I = 0, N = Number Of Elements, Key = Search Value
        3)  Read Value Of Key To Be Searched I.E. "Key"
        4)  If K[I] == Key
                Print "Record At Position I"
                Go To Step 8
        5)  Increment I I.E. I++
        6)  If I < N
                Go To Step 4
        7)  Print "Element Not Found"
        8)  Stop
