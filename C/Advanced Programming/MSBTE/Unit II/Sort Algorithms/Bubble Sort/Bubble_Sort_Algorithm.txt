Bubble Sort:

    Algorithm:

        1)  Read The Total Number Of Elements Say 'N'
        2)  Store The Element In The Array. Set P = 1, Pass = N- 1
        3)  Set I = 0
        4)  Compare Two Adjacent Elements.
                I.E. A[I] With A[I + 1]
        5)  If A[I] > A[I + 1] Then Interchange Both Elements.
        6)  Increment Value Of I    I.E. I++
        7)  Repeat Step 4 And 5 For I < N - 1
        8)  Increment P     I.E. P++
        9)  If P <= Pass Then Go To Step No.3
        10) Print The Sorted Array
        11) Stop
