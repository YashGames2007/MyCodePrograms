Insertion Sort:

    Algorithm:

        1)  Start
        2)  First Iteration Is Compared With 0th Element If It Is Smaller It
            Is Inserted In The 0th Place Of Sorted Array.
        3)  Compare The Remaining Elements One By One And Place All Elements 
            In Proper Position Using Shift Insert Function.
        4)  Shift All The Elements In The Sorted Sub-List That Is Greater That The Value To Be Sorted
        5)  Repeat Until List Is Sorted
        6)  Stop
