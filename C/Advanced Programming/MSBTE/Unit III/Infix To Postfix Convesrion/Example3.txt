>>==Question==>>    Convert The Infix Expression To Postfix.           

>==>>               #{1}#   ( A + B ) * C

>>==Solution==>>

+----------+---------+-----------+-----------+
|  Symbol  |  Stack  |  Postfix  |  Applied  |
|   Scan   |         |           |    Rule   |
+----------+---------+-----------+-----------+
|    (     |    (    |   Empty   |    R2     |
+----------+---------+-----------+-----------+
|    A     |    (    |     A     |    R1     |
+----------+---------+-----------+-----------+
|    +     |   ( +   |     A     |    BR2    |
+----------+---------+-----------+-----------+
|    B     |   ( +   |    A B    |    R1     |
+----------+---------+-----------+-----------+
|    )     |  Empty  |   A B +   |    BR3    |
+----------+---------+-----------+-----------+
|    *     |    *    |   A B +   |    R2     |
+----------+---------+-----------+-----------+
|    C     |    *    |  A B + C  |    R1     |
+----------+---------+-----------+-----------+
|  Empty   |         | A B + C * |    PR4    |
+----------+---------+-----------+-----------+

>>===Answer==>>

#***#   The Postfix Expression Is: A B + C *.  #***#
